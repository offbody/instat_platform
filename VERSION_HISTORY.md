
# История версий: Инстат MVP

**Название релиза:** Инстат MVP (Стабильный снимок)
**Версия:** 1.0.0
**Дата:** 24 мая 2024 г.
**Статус:** Базовая функциональность завершена

## Обзор реализованных возможностей

Данный резервный снимок представляет собой готовую MVP-версию платформы **Инстат** — высокопроизводительного решения для управления ESG-показателями и Стандартом общественного капитала бизнеса (СОКБ).

### 1. Центральный Дашборд (Обзор)
- **Матрица устойчивости**: Интерактивные радарные диаграммы для анализа 5 векторов (Экономика, Персонал, Общество, Экология, Государство).
- **KPI-карточки**: Статистические блоки с динамическими графиками трендов (Recharts).
- **Анализ производительности**: Комбинированные графики (План/Факт) и расчет эффективности.
- **Структура закупок**: Мониторинг локализации и доли закупок у МСП.

### 2. ИИ-Интеллект (Gemini API)
- **Стратегические отчеты**: Автоматическая генерация глубокой аналитики на основе текущих данных дашборда.
- **ИИ-Ассистент**: Контекстный чат-виджет для оперативных ответов на вопросы по устойчивому развитию.
- **Прогнозирование**: Расчет прогнозных значений до 2027 года с учетом CAGR.

### 3. Анкета СОКБ
- **Гибкие режимы**: Поддержка «Полного» и «Базового» перечней вопросов.
- **Категоризация**: Разделы «Здоровье», «Экология», «Общество», «Экономика».
- **Валидация**: Интерфейс для загрузки подтверждающих документов и отслеживания прогресса сертификации.

### 4. Региональное влияние
- **Интерактивная карта**: Интеграция с Yandex Maps API для визуализации присутствия в регионах.
- **Карточки регионов**: Детализация инвестиций, количества социальных проектов и индекса влияния.

### 5. Экосистема экспертов
- **База специалистов**: Каталог верифицированных экспертов Академии СОКБ с системой фильтрации.
- **Коллаборация**: Возможность подключения экспертов к конкретным разделам дашборда для аудита.
- **Прямая связь**: Профили с контактами и быстрым доступом к мессенджерам.

### 6. Технический стек и UI
- **Дизайн-система**: Адаптивный интерфейс в стиле Atlassian (Inter Design).
- **Производительность**: Оптимизированные React-компоненты с мемоизацией.
- **Доступность**: Поддержка темной и светлой тем с сохранением состояния.
